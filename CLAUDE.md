# CLAUDE.md

AIアシスタント（Claude Code等）向けのガイドラインです。

## 概要

最小構成の静的HTMLプロジェクト。ファイルは `index.html` のみで、ビルドツール・依存関係・フレームワークは一切なし。

## 構成

```
/
├── index.html   # HTML5シングルページ（プレースホルダーのみ）
└── CLAUDE.md    # このファイル
```

## コード規約

| 項目 | ルール |
|---|---|
| HTML標準 | HTML5（`<!DOCTYPE HTML>`） |
| 言語属性 | `<html lang="...">` を必ず設定 |
| 文字コード | `<head>` 内に `charset="UTF-8"` を宣言 |
| インデント | タブ |
| 外部リソース | 必要な場合のみ追加 |

## 開発ワークフロー

- デフォルトブランチ: `master`
- AIブランチ形式: `claude/<説明>-<セッションID>`
- 作業前に正しいフィーチャーブランチにいることを確認する
- `master` への直接プッシュは禁止（明示的許可がある場合を除く）
- コミット: 命令形・簡潔に（例: `Add navigation bar`）
- プッシュ: `git push -u origin <ブランチ名>`
- フォースプッシュ・リベース・`--no-verify` は明示的指示がある場合のみ

## ビルド・テストツール

設定なし。`npm` / `yarn` / `pnpm` は実行しない。設定ファイルも依頼なしに作成しない。

## AIアシスタントへの注意

- 変更は依頼内容に絞った最小限にとどめる
- フレームワーク・ライブラリの追加はユーザー承認が必要
- 不要なファイルを作成しない
- スタイル・スクリプトはインライン（`<style>` / `<script>`）を優先
- テスト・サンドボックス環境として扱う
